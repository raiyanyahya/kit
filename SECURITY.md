# Security Policy

## Supported Versions

Only the latest release of Kit receives security updates.

## Reporting a Vulnerability

If you find a security vulnerability in Kit, please do not open a public GitHub issue. Email me directly at **raiyanyahyadeveloper@gmail.com** with the details.

Include as much as you can:
- What the vulnerability is and where it exists
- Steps to reproduce it
- What impact it could have

I'll acknowledge your report within 48 hours and keep you updated as I work on a fix. Once a patch is released I'm happy to credit you in the release notes if you'd like.
